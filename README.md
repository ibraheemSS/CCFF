# Conditional-Capture Flip-Flop (CCFF)

Design and implementation of a low-power **Conditional-Capture Flip-Flop (CCFF)** using **Electric VLSI**.

---

## 📖 Overview

This project presents the design and implementation of a **Conditional-Capture Flip-Flop (CCFF)**, a low-power sequential circuit that reduces unnecessary switching activity to improve energy efficiency while maintaining reliable performance.

The project was developed using **Electric VLSI** and includes the complete circuit implementation, project paper, presentation, and simulation results.
